- seaborn
- sci-kit learn
- tqdm

## To Run:
- mkdir data
- Download .csv files from kaggle (link above) into data
- pip install pipenv
- pipenv install jupyter lab
- pipenv install -r requirements.txt
- pipenv run jupyter lab
